ODE

Ordinary Differential Equation:

  • any equation of ordinary derivative
  • way of modeling continuous change

Hybrid programs

Syntax:

  • α,β are programs
  • x:=θ is discrete assignment
  • x′=f(x)&Q − let x′ evolve while Q is true
  • ?Q − abort if Q is not true
  • α∪β − nondeterministically run α or β
  • α;β − run β in state resulting from α
  • α* - repeat α any nonnegative number of times

See Bohrer, Luo, Chuang, Platzer 2018
